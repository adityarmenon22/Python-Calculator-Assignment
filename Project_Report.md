# Project Documentation

## Project Overview
This project is a CLI (Command Line Interface) calculator designed to perform basic mathematical operations efficiently.

## Development Process
1. **Setup:** Initialized a Git repository in VS Code.
2. **Logic:** Implemented functions for arithmetic operations.
3. **Interface:** Created a while-loop to accept user input continuously.
4. **Testing:** Verified edge cases like division by zero.

## Future Improvements
- Add a Graphical User Interface (GUI) using Tkinter.
- Add advanced math functions (square root, power).
